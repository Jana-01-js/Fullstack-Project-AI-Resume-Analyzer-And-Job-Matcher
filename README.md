# AI Resume Analyzer

A Full Stack Flask Application that allows users to register, log in, upload resumes, and receive an ATS score based on detected technical skills.

## Features

- User Registration
- User Login & Logout
- Resume Upload
- ATS Score Calculation
- Skill Extraction
- SQLite Database
- Responsive Interface

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install flask werkzeug
```

### Run Application

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
